Kevin's Car Project A car I made in Computer Science

For this project, I made a car using a program called Tinkercad. I used this software to not only model this car, but also another house that I made. A cool thing about this project is that I incorporated Arduino parts to this car so it can actually work when you print it. I used the same skills from my house project, but this time scaling was a bigger problem because of the Arduino parts. I learned how to incorporate Arduino parts in this project.
