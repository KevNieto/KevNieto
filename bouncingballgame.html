Kevin's Bouncing Ball game A simple game I made in Computer Science

In this project, I used a program called Processing in order to code a game that is similar to the game Pong but it only has one paddle. Because of that, I changed the name of the game to "Bouncing Ball game" since it can't really be classified as Pong. The game consists of bouncing a ball off a paddle to earn points and if the ball goes behind the paddle, the game resets. Below you can see a screenshot of the game.
